# Algoritma
Step 1 : jika n = 0 return nilai m sebagai hasil dari stop. Jika tidak, kembali ke step 2
Step 2 : bagi nilai m dan n, masukan nilai sisanya ke 1
Step 3 : masukan nilai n ke m dan nilai r ke n. Kembali ke step 1

Pseudocode :
While n ≠ 0 do
r < m mod n 
M < n
N < r
Return m
